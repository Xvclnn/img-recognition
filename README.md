# img-recognition

Өгөгдсөн directory дотроос зорилтот царайтай хүн орсон зургуудыг хайна.

### Төслийн бүтэц

- `config.py` — Тохиргооны нэгдсэн файл (модел, threshold, файлын замууд гэх мэт).
- `build_db.py` — Зургуудаас царайны вектор гарган авч, индексжүүлсэн `.pkl` файл үүсгэнэ.
- `face_recognize.py` — Матриц үржвэр ашиглан (`O(N)`) ижил царайг хугацааны алдагдал маш багатай хайна.
- `requirements.txt` — Шаардлагатай сангуудын жагсаалт.

---

## Суулгах заавар

```bash
# 1. Төслийн хавтас руу шилжих
cd img-recognition

# 2. Virtual environment үүсгэж, идэвхжүүлэх
python -m venv .venv
source .venv/bin/activate

# 3. Шаардлагатай сангуудыг суулгах
pip install -r requirements.txt
```

---

## Ажиллуулах заавар

### 1. Тохиргоо хийх

`config.py` файл дотор модел болон зургийн directory замыг тохируулна.

Өгөгдсөн модел болон аргачлалд тохируулж threshold утгыг дараах байдлаар тохируулсан болно.

```python
THRESHOLD = 0.30
```

---

### 2. .plt файл үүсгэх

```bash
python pkl.py
```

> Царай ороогүй зургуудыг систем автоматаар алгасаж, индексжүүлсэн `.pkl` файл үүсгэнэ.

---

### 3. Ижил царай хайх

`face_recognize.py` файл дотор хайх зургийн замыг (`HAIH_ZURAG`) тохируулсны дараа дараах командыг ажиллуулна:

```bash
python face_recognize.py
```

> Хэрэв зурган дээр олон хүн байвал систем царай бүрийг тусад нь илрүүлж, .pkl-тай харьцуулна. `Distance` утга нь `THRESHOLD`-оос бага байгаа бүх илэрцийг Bounding Box-ийн мэдээлэлтэй нь хэвлэнэ.
